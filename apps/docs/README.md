# Documentation

[Mintlify](https://mintlify.com) documentation site. Runs on port 3004. Each page is an MDX file with components, search, and syntax highlighting.

## Development

```bash
mint dev
```

## Key Features

- MDX pages with Mintlify components
- API reference with OpenAPI support
- AI tools guides (Claude Code, Cursor, Windsurf)
- Configuration via `docs.json`

Pages live in the root of this directory. Add new pages to `docs.json` navigation.

## Publishing

Install the [Mintlify GitHub app](https://dashboard.mintlify.com/settings/organization/github-app) to auto-deploy on push.
