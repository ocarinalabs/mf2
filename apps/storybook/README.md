# Storybook

Frontend workshop for the design system. Runs on port 6006.

## Development

```bash
turbo dev --filter=storybook
```

## Key Features

- All shadcn/ui components with stories
- Dark mode support via `@storybook/addon-themes`
- Chromatic visual testing integration
- Tailwind CSS 4 styling

Components are imported from `@repo/design-system`. Add stories in `stories/`.
