# Email

Email templates using React Email. Runs on port 3003.

## Development

```bash
turbo dev --filter=email
```

## Key Features

- React Email preview server with hot reload
- Pre-built contact form template
- Resend integration via `@repo/email`

Templates live in `packages/email/templates/`. The preview server renders them at `localhost:3003`.
