# @repo/convex

React provider that connects Convex to Clerk authentication.

## Usage

```ts
import { ConvexClientProvider } from "@repo/convex/provider";
```

## Environment Variables

| Variable | Required | Description |
|----------|----------|-------------|
| `NEXT_PUBLIC_CONVEX_URL` | No | Convex deployment URL (web) |
| `EXPO_PUBLIC_CONVEX_URL` | No | Convex deployment URL (mobile) |
