# @repo/collaboration

Real-time collaboration with Liveblocks. Live cursors, avatar stacks, and presence indicators.

## Usage

```ts
import { CollaborationProvider } from "@repo/collaboration/room";
```

## Environment Variables

| Variable | Required | Description |
|----------|----------|-------------|
| `LIVEBLOCKS_SECRET` | No | Liveblocks secret key |
