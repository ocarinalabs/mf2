# @repo/webhooks

Outbound webhook delivery with Svix.

## Usage

```ts
import { sendWebhook } from "@repo/webhooks";
```

## Environment Variables

| Variable | Required | Description |
|----------|----------|-------------|
| `SVIX_TOKEN` | No | Svix API token |
