# @repo/typescript-config

Shared TypeScript configurations for all apps and packages.

## Usage

```json
{
  "extends": "@repo/typescript-config/nextjs.json"
}
```

Presets: `base.json`, `nextjs.json`, `react-library.json`, `convex.json`, `expo.json`.
