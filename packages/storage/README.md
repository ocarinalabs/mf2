# @repo/storage

File upload and storage with Convex and Vercel Blob.

## Usage

```ts
import { put } from "@repo/storage/client";
```

## Environment Variables

| Variable | Required | Description |
|----------|----------|-------------|
| `BLOB_READ_WRITE_TOKEN` | No | Vercel Blob read/write token |
