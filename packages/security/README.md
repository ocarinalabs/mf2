# @repo/security

Bot detection, rate limiting, and secure headers with Arcjet and Nosecone.

## Usage

```ts
import { securityMiddleware, noseconeOptions } from "@repo/security/proxy";
```

## Environment Variables

| Variable | Required | Description |
|----------|----------|-------------|
| `ARCJET_KEY` | No | Arcjet API key |
