# @repo/feature-flags

Feature flag management for gradual rollouts.

## Usage

```ts
import { showBetaFeature } from "@repo/feature-flags";
```

## Environment Variables

| Variable | Required | Description |
|----------|----------|-------------|
| `FLAGS_SECRET` | No | Secret for flag evaluation |
